# 6. 모의고사(실전형) 구성안

> **주의:** 실제 기출문제 원문은 공개되지 않는 경우가 많습니다. 아래는 **후기 기반 유형**을 반영한 **모의 실전형 구성안**입니다.

## 모의고사 1회차 (180분 / 3문항)
- **문항 1 | Tabular 분류/회귀**
  - 목표: 불균형 데이터에서 F1/Recall 최적화
  - 핵심: 전처리(결측/이상치) → baseline → 튜닝
- **문항 2 | 텍스트 분류**
  - 목표: TF-IDF+ML vs Tokenizer+DL 비교
  - 핵심: 불용어 처리, n-gram, class_weight
- **문항 3 | 이미지 분류**
  - 목표: 전이학습 모델 성능 최적화
  - 핵심: Augmentation, freeze/unfreeze 전략

## 모의고사 2회차 (180분 / 3문항)
- **문항 1 | Tabular 분류**
  - 목표: ROC-AUC 또는 F1 최적화
  - 핵심: 피처 엔지니어링, 앙상블
- **문항 2 | 텍스트 다중분류**
  - 목표: 불균형 클래스 대응
  - 핵심: threshold 조정, macro F1
- **문항 3 | 이미지 분류**
  - 목표: 과적합 방지
  - 핵심: 데이터 증강, 규제, learning rate 스케줄링

# 7. 실전문제(기출 유형) 포인트 정리

> 실제 기출의 핵심은 **문제 구조와 풀이 흐름**입니다. 아래 포인트를 익숙하게 만들어두면 실전에서 속도가 붙습니다.

- **Tabular**: 결측/이상치 처리 → 인코딩 → baseline → 튜닝 → 지표 개선
- **Text**: 전처리(불용어/토큰화) → 벡터화 → 모델 비교 → 성능 개선
- **Image**: 전이학습 → 데이터 증강 → 과적합 방지 → 정확도 최적화
- **공통**: 지표 선택 이유를 설명할 수 있어야 함

# 8. 예상문제 15개 + 풀이 방향

### 1) 불균형 분류 데이터
**Q:** 이탈률 5% 데이터에서 적합한 지표는?  
**A:** Accuracy는 의미 없음 → **Recall / F1 중심**

### 2) 회귀 오차 큼
**Q:** RMSE가 크다면?  
**A:** 이상치 처리, 로그변환, 피처 개선, 모델 변경

### 3) 과적합
**Q:** Train↑ Test↓  
**A:** 규제, 교차검증, 모델 단순화

### 4) 결측치 비율 높음
**Q:** 대응 전략?  
**A:** 비율 판단 후 제거/대체/예측 모델

### 5) 범주형 변수 많음
**Q:** 인코딩 전략?  
**A:** One-hot vs Target encoding 선택

### 6) 텍스트 분류 baseline
**Q:** 가장 빠른 baseline은?  
**A:** TF-IDF + Logistic Regression

### 7) 텍스트 성능 개선
**Q:** 성능 낮을 때?  
**A:** n-gram, 불용어 처리, 사전학습모델

### 8) 이미지 분류 성능 개선
**Q:** 성능 낮을 때?  
**A:** 전이학습 + 데이터 증강

### 9) 다중 클래스 불균형
**Q:** 대응 전략?  
**A:** class_weight, oversampling

### 10) 피처 중요도 요구
**Q:** 해석 방법?  
**A:** SHAP / Permutation

### 11) 시간 부족 문제
**Q:** 시간 관리?  
**A:** Baseline 1~2개 확보 후 개선

### 12) 스케일링 필요 여부
**Q:** 어떤 모델에서 중요?  
**A:** 선형/거리 기반 모델

### 13) 성능 향상 우선순위
**Q:** 무엇부터 개선?  
**A:** 데이터 품질 → 피처 → 모델

### 14) Tabular 분류 baseline
**Q:** 가장 빠른 모델?  
**A:** Logistic Regression / RandomForest

### 15) 검증 방식
**Q:** 어떤 검증이 적절?  
**A:** Stratified K-Fold
